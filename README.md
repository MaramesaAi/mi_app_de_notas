# 📝 Mi App de Notas

Una aplicación web minimalista para escribir notas de forma rápida y sin complicaciones. Solo HTML y CSS (con un poquito de JavaScript vanilla).

## 🚀 ¿Qué hace?

- Abrís la app y empezás a escribir tu nota en un área de texto grande y cómoda.
- Un contador de caracteres te muestra cuánto llevas escrito.
- Un botón de **Limpiar** borra todo el contenido para que empieces de nuevo.

## ⚠️ Importante

Las notas **no se guardan**. Si cerrás la pestaña o el navegador, la nota se pierde. Esta app está pensada como un bloc de notas efímero, ideal para anotar algo rápido mientras trabajás.

## 📁 Estructura del repositorio

```
mi_app_de_notas/
├── index.html    →  La aplicación principal (estructura + lógica)
├── style.css     →  Todos los estilos visuales
└── README.md     →  Este archivo
```

## 🛠️ Cómo usar

1. Cloná o descargá el repositorio:
   ```bash
   git clone https://github.com/MaramesaAi/mi_app_de_notas.git
   ```
2. Abrí `index.html` en tu navegador favorito.
3. ¡Escribí tu nota!

También podés abrirlo directamente desde GitHub Pages si está habilitado.

## 🎨 Tecnologías

- **HTML5** — estructura semántica
- **CSS3** — diseño responsive con gradientes y sombras
- **JavaScript vanilla** — contador de caracteres y botón de limpiar (sin dependencias)
